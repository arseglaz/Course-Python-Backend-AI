1) Один элемент - `.append()`, несколько элементов из другого списка - `.extend()`

2) Оба удаляют элемент по индексу, но `.pop()` возвращает значение, то есть можно записать удаленное значение в переменную и потом использовать

3) Вопрос про ссылки, список - конкретный обьект, на него могут ссылаться несколько переменных, тогда значение обьекта (списка) меняется. 
Но можно сделать копию списка, тогда будет два разных обьекта. 
```python

second_list = main_list.copy()
second_list.append("Vortex")

print(main_list)
print(second_list)
```
